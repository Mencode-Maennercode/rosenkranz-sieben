# Manuela Rosenkranz Praxis - Redesign 2025

## Übersicht
Neues Homepage-Design mit sanften, beruhigenden Farben und modernem Layout.

## Design-Philosophie
Das neue Design wurde inspiriert von:
- https://praxis-charis.de/
- https://schildhauer-hormontherapie.de/

### Farbschema
**Weiche, natürliche Farbtöne:**
- **Salbeigrün** (#a7b9a4) - Hauptfarbe, beruhigend und natürlich
- **Beige** (#e8dfd6) - Warm und einladend
- **Creme** (#faf8f5) - Hintergrundfarbe, hell und luftig
- **Sekundärfarbe** (#d4c5b9) - Warme Akzente

Diese Farben wurden gewählt, um:
- Eine beruhigende, professionelle Atmosphäre zu schaffen
- Vertrauen und Wärme zu vermitteln
- Eine angenehme Lesbarkeit zu gewährleisten

## Features

### 1. **Modernes, responsives Layout**
- Mobile-first Design
- Optimiert für alle Bildschirmgrößen
- Hamburger-Menü für mobile Geräte

### 2. **Inhalte**
- **Hero-Bereich**: Einladende Willkommensbotschaft mit sanftem Hintergrundbild
- **Über mich**: Vollständige Biografie mit:
  - Geboren 1967 in Deutschland
  - Wohnhaft in Bad Neuenahr-Ahrweiler
  - Verheiratet seit 1991, 2 erwachsene Söhne
  - Qualifikationen (Finanzwirtin 1987, Individualpsychologische Beraterin 2025, Seelsorgerin 2025)
- **Leistungen**: 4 Hauptbereiche mit Bildern
- **Preise**: Transparente Preisübersicht
- **Mein Ansatz**: Individualpsychologie nach Alfred Adler
- **Kontaktformular**: Einfache Kontaktaufnahme

### 3. **Professionelle Bilder**
- Alle Bilder von Unsplash (lizenzfrei)
- Psychologische, beruhigende Motive
- Keine sichtbaren Personen (außer Platzhalter für Ihr Foto)
- Naturmotive und abstrakte Szenen

### 4. **Smooth Animations**
- Sanfte Scroll-Effekte
- Fade-in Animationen beim Scrollen
- Hover-Effekte auf Karten
- Smooth Scroll Navigation

### 5. **Barrierefreiheit**
- Semantisches HTML
- Tastaturnavigation
- Responsive Design

## Technische Details

### Schriftarten
- **Montserrat**: Moderne, klare Sans-Serif für Fließtext
- **Crimson Pro**: Elegante Serif für Überschriften

### Browser-Kompatibilität
- Chrome, Firefox, Safari, Edge (neueste Versionen)
- Mobile Browser (iOS Safari, Chrome Mobile)

## Installation & Nutzung

1. Öffnen Sie `index.html` in einem Webbrowser
2. Für lokales Testen: 
   - Verwenden Sie einen lokalen Webserver
   - Oder öffnen Sie die Datei direkt im Browser
